Per-client artefact folder

Create clients/<name>/ or docs/artefacts/<name>/:

  • Discovery answers (markdown or sheet)
  • GST / Udyam PDFs (never commit secrets to public git)
  • Formspree + GA4 IDs in README or private vault
  • Progress update HTML for client meetings
  • Signed catalogue approval (WhatsApp screenshot OK)
